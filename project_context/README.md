# 项目背景资料说明

本目录用于存放后续大模型接手 `RL/` 实验项目时需要阅读的 PDF 背景资料。它们不是训练脚本，也不是实验输出，而是帮助模型理解毕业设计目标、论文叙事、已有方法和当前工程任务的上下文材料。

## 文件清单

| 文件 | 作用 |
|---|---|
| `本科毕业论文_完整定稿.pdf` | 当前毕业论文完整定稿版本，用于理解课题背景、方法描述、实验口径和论文叙事。 |
| `本科毕业论文_LaTeX当前编译版_main.pdf` | LaTeX 当前编译版，用于和定稿版本交叉确认论文内容。 |
| `Pentest-R1论文_2508.07382v2.pdf` | 主要技术参考论文，用于理解两阶段强化学习、Web/CTF 渗透测试任务、AutoPenBench/CyBench 评测和 token 代价分析。 |
| `本科毕设任务书.pdf` | 毕设任务来源，用于确认题目、研究目标和工作边界。 |
| `本科毕设开题报告.pdf` | 开题阶段方案，用于理解早期研究动机、技术路线和预期成果。 |
| `本科毕设中期进展检查表.pdf` | 中期阶段进展，用于理解已完成工作、实验计划和论文主要内容。 |
| `本科毕设中期答辩.pdf` | 中期答辩材料，用于快速理解项目动机、方法设计和实验展示口径。 |

## 推荐阅读顺序

1. 先读 `../当前RL实验后续待办清单.md`，明确当前任务优先级。
2. 再读 `../PENTEST_R1_JUDGE_EXPERIMENT_FLOW.md`，理解工程链路和 Judge 进展奖励设计。
3. 再读 `本科毕业论文_完整定稿.pdf` 和 `本科毕业论文_LaTeX当前编译版_main.pdf`，理解论文整体叙事。
4. 再读 `Pentest-R1论文_2508.07382v2.pdf`，理解技术背景和原始实验口径。
5. 最后按需参考任务书、开题、中期材料。

## 当前最重要结论

当前工程阶段的核心目标是先跑通训练，不是先追求论文正式指标。后续大模型应优先遵循 `../当前RL实验后续待办清单.md`：先使用 `purpcode/rl/main_grpo_pentest_r1_stage2_quality_speed.sh` 跑完一次 quality-speed 训练闭环，再考虑 strict no-judge repro、Judge 训练和 AutoPenBench 正式评测。
